## Run Locally

**Prerequisites:**  Node.js


1. Instal dependensi:
   `npm install`
2. Atur `GEMINI_API_KEY` di [.env.local](.env.local) dengan API key Gemini Anda
3. Jalankan aplikasi:
   `npm run dev`
